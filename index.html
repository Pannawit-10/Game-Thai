🎯 คำประสม (Compound Words)

นิยาม: คำที่เกิดจากการนำคำสองคำมารวมกัน โดยมีความหมายใหม่ที่แตกต่างจากคำเดิม

ตัวอย่างคำประสม

  • น้ำแข็ง = น้ำ + แข็ง
  • ดินสอ = ดิน + สอ
  • รถไฟ = รถ + ไฟ
  • หนังสือ = หนัง + สือ
  • แสงแดด = แสง + แดด

คุณสมบัติสำคัญ

  • ✅ มีความหมายใหม่จากการรวมกัน
  • ✅ ใช้ในชีวิตประจำวันบ่อย
  • ✅ เข้าใจง่าย ไม่ซับซ้อน
  • ✅ มักเป็นคำนาม

👑 คำสมาส (Royal/Formal Compound)

นิยาม: คำที่มีลักษณะทางการ มักใช้ในราชการ หรือเป็นคำที่มีความเป็นทางการสูง

ตัวอย่างคำสมาส

  • ราชบุตร = ลูกชายของกษัตริย์
  • มหาเทพ = เทพที่ยิ่งใหญ่
  • อธิบดี = หัวหน้าระดับสูง
  • รัฐมนตรี = รัฐ + มนตรี
  • อธิการบดี = หัวหน้ามหาวิทยาลัย

คุณสมบัติสำคัญ

  • ❌ เป็นคำทางการ
  • ❌ ใช้ในราชการ/องค์กร
  • ❌ มีความเป็นทางการสูง
  • ❌ ไม่ใช่คำประสมธรรมดา

🔄 คำซ้อน (Reduplication)

นิยาม: คำที่เกิดจากการนำคำเดียวกันมาซ้ำ เพื่อเน้นความหมายหรือแสดงระดับ

ตัวอย่างคำซ้อน

  • เด็กเด็ก = เด็กมาก
  • สวยสวย = สวยมาก
  • เก่าเก่า = เก่ามาก
  • ใหม่ใหม่ = ใหม่มาก
  • ร้อนร้อน = ร้อนมาก

คุณสมบัติสำคัญ

  • ❌ ซ้ำคำเดียวกัน
  • ❌ เน้นความหมาย
  • ❌ แสดงระดับความ
  • ❌ ไม่ใช่การรวมคำต่างกัน

🌱 คำมูล (Root Words)

นิยาม: คำพื้นฐานที่ไม่ได้เกิดจากการรวมคำอื่น เป็นคำเดี่ยวที่มีความหมายในตัวเอง

ตัวอย่างคำมูล

  • กิน = การบริโภคอาหาร
  • ดื่ม = การบริโภคเครื่องดื่ม
  • นอน = การพักผ่อน
  • เดิน = การเคลื่อนไหว
  • ดู = การมอง

คุณสมบัติสำคัญ

  • ❌ เป็นคำเดี่ยว
  • ❌ ไม่ได้รวมจากคำอื่น
  • ❌ เป็นคำพื้นฐาน
  • ❌ มักเป็นกริยา คุณศัพท์

🔗 คำสมาสสนธิ (Sandhi Compound)

นิยาม: คำที่เกิดจากการรวมคำโดยมีการเปลี่ยนแปลงเสียงหรือรูปแบบ มักเป็นคำทางวิชาการ

ตัวอย่างคำสมาสสนธิ

  • สุริยุปราคา = สุริยะ + อุปราคา
  • วิทยาศาสตร์ = วิทยา + ศาสตร์
  • คณิตศาสตร์ = คณิต + ศาสตร์
  • ปฐมนิเทศ = ปฐม + นิเทศ
  • มหาวิทยาลัย = มหา + วิทยาลัย

คุณสมบัติสำคัญ

  • ❌ มีการเปลี่ยนแปลงเสียง
  • ❌ เป็นคำทางวิชาการ
  • ❌ ซับซ้อนกว่าคำประสม
  • ❌ มักมาจากภาษาบาลี สันสกฤต

📝 สรุปสำคัญ
